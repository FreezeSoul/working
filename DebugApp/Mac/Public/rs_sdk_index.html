SDK 入门教程

Romanysoft SDK 提供两个重要的js文件来调用macOS原生功能 及 扩展插件。它们是 bs.js 和 util.js

bs.js 和 util.js 依赖 JQuery. （兼容的JQuery版本从1.7 到 JQuery 3.X ）

可以从这里获取兼容版本: https://github.com/Romanysoft/working/sdkjs/

备注:JQuery2.x - JQuery3.x 引入工程中，要引用相应版本的jquery-migrate插件



  • bs.js 主要提供封装好的方法来调用 macOS 原生功能和扩展插件.
  • util.js 主要提供一些方便的工具类函数

使用JQuery 1.9 版本

使用JQuery 3.1.1 版本

util.js

是在JQuery的基础上进行扩展的工具函数, 直接通过 $. 或 JQuery 就可以访问

涉及很多共用的处理函数，是SDK重要的组成部分。完成的API使用说明，你可以点击下面的链接来查看: Utils

bs.js

是调用macOS原生功能及扩展插件的入口。 在window对象上，注册了BS 属性。一般，直接使用 BS.b$ 就可以直接访问内置的功能.

完成的API使用说明，请参照以下链接：
